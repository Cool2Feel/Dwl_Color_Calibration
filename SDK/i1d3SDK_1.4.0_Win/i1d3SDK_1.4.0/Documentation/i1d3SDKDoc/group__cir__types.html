Enumeration Type Documentation

anonymous enum

Measurement control states for backlight synchronization.

An enumeration that describes the available options that allow to choose between a normal measurement and a measurement that is synchronized to the display's backlight frequency. In some cases a synchronized measurement might yield better results.
